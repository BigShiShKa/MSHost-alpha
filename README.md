# Описание
Система для управления Minecraft-сервером с веб-панелью управления. Позволяет запускать, останавливать и мониторить Minecraft-сервер через удобный веб-интерфейс или консольные команды.

# Основные возможности
Управление сервером
Запуск/остановка Minecraft-сервера

Перезапуск сервера

Отправка команд на сервер

Мониторинг статуса сервера

# Веб-панель
REST API для управления

Просмотр логов в реальном времени

Загрузка модпаков

Аутентификация по токенам

Ограничение скорости загрузки

Консольный интерфейс

Интерактивные команды

Автодополнение команд

Полный контроль через терминал

# Требования
Системные требования
Windows

Linux (в разработке)

Компилятор C++17 (GCC/Clang/MSVC)

CMake 3.12+

# Библиотеки:
nlohmann-json, cpp-httplib

# Зависимости
Winsock2 (Windows)

Потоки C++11

Файловая система C++17
